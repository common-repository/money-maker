== Installation ==
= For an automatic installation through WordPress: =
1. Go to the 'Add New' plugins screen in your WordPress admin area
2. Search for 'money maker'
3. Click 'Install Now'
4. Activate the plugin
		
= For a manual installation via FTP: =
1. Upload the money-maker folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area
3. start blogging
		
= To upload the plugin through WordPress, instead of FTP: =
Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate it.
